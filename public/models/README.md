## ML Models

This directory contains face-api.js model files that will be automatically downloaded from CDN when the application first loads.

### Models Used:
- **tiny_face_detector_model** (~190KB) - Fast face detection
- **face_landmark_68_model** (~350KB) - 68 facial landmark points  
- **face_recognition_model** (~6.2MB) - 128-dimensional face descriptors

Models are served from: https://github.com/justadudewhohacks/face-api.js-models

The models will be loaded at runtime from this directory.
